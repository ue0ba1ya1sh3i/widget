## このアプリについて
- このアプリはTauriとReactなどの技術を使用して作られたシンプルで高速なウィジェットアプリです。
![スクリーンショット](./screenShot/japanese.png)

## セットアップ
- `npm install`を実行してください

## 使い方
- 開発モードで立ち上げる時
  - `npm run tauri dev`を実行してください

- アプリをビルドするとき
  - `npm run tauri build`を実行してください
